# NestJS GraphQl application for jokes

## Installation

```bash
$ npm install
```

## Running the app

```bash
# if your default nodeJS version is not v14.17.1
$ nvm use 

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
